## 一、任务来源与领域背景

本次个人分析任务来源于全球海洋大气耦合时空可视分析系统项目，围绕海洋学与气候科学领域的真实研究需求设计。通过文献调研发现，**全球海表温度（SST）异常的时空演变规律**是当前海洋气候研究的核心热点之一，具体体现在以下三个方向：

- **ENSO事件检测与分类**：CP型与EP型El Niño的空间分布差异及频率年代际变化（Atmosphere, 2025）
- **海洋热含量异常分析**：OHC在不同海域的积累与释放规律，与ENSO事件的时间相关性
- **海气耦合相关性**：SST距平与大气风应力之间的双向反馈机制（Bjerknes反馈）

基于上述热点，本人将个人分析任务确定为：**全球海表温度异常的时空演变规律分析**，重点关注ENSO事件期间SST距平的空间扩散模式、OHC海域对比及海气耦合特征。

## 二、大分析任务定义

**任务名称：** 全球海表温度异常时空演变规律分析

**分析目标：** 面向海洋科研人员与气候分析用户，支持对全球6个典型海域（赤道太平洋、北太平洋、印度洋、大西洋等）在多年月尺度上的SST距平演变进行探索、定位与查询，识别ENSO事件的空间扩散模式及其与OHC、风应力之间的耦合关联。

**数据来源：** ERSSTv5月均SST异常数据 + ERA5再分析风应力数据，共3600条记录（6海域 × 36月），字段包括经纬度、时间、SST距平、海洋热含量OHC、风应力、盐度等。

## 三、任务抽象（Brehmer & Munzner 三级分类）

### 3.1 Analyze 层（高级）— 发现规律

**任务描述：** 分析ENSO暖事件期间SST距平的全球空间扩散模式，以及SST距平与大气风应力的时空耦合规律。

**领域依据：** 研究表明1990年以来赤道西太平洋SSTA中心向西偏移约18个经度，且CP型El Niño频率显著上升，这一空间演变规律需要通过时序动画与联动散点图才能有效呈现。

**实现方式：**
- 地理热力图配合时间轴播放，观察SST距平的空间扩散过程
- 散点图（SST距平 × 风应力，颜色编码海域）揭示海气耦合强度
- 折线图辅助判断当前时段的ENSO演变阶段

---

### 3.2 Search 层（中级）— 定位特征

**任务描述：** 在指定海域与时间范围内，探索OHC异常高值月份，定位ENSO事件的峰值时段与空间中心。

**领域依据：** ENSO事件强度检测通常以ONI指数连续5个月超过±0.5℃为判定标准，对应可视化中需要支持按阈值筛选与异常月份快速定位。

**实现方式：**
- 区域筛选（命名海域 / 空间bbox）触发接口 `/api/data/filter`
- 柱状图按海域展示月均OHC，排序后直观识别异常高值
- 折线刷选时间段后，柱状图与散点图联动同步更新
- Tooltip悬浮显示具体数值与偏差

---

### 3.3 Query 层（低级）— 识别单点

**任务描述：** 查询指定经纬度站点在某月份的SST值与历史气候态偏差，判断该格点是否处于异常状态。

**领域依据：** 业务化海洋监测中，科研人员需要快速确认特定站位的当前异常状态，是最基础也是最高频的操作需求。

**实现方式：**
- 地图点击站点 → 调用 `/api/data/point` 接口
- 右侧详情面板展示：SST、SST距平、OHC、风应力，以及与同海域历史均值的偏差
- 自动标注当前是否满足El Niño / La Niña判定阈值（±0.5℃）

## 四、界面布局与视觉编码

个人独立分析界面采用四图联动布局：

```
┌──────────────────────────────────────────────┐
│   顶部：全局时间轴（双滑块 + ENSO预设 + 播放）     │
├─────────────────┬────────────────────────────┤
│  ① 折线图        │  ② 地理热力图（主图）         │
│  各海域SST距平   │  全球SST距平空间分布           │
│  时间趋势        │  发散色标 · 点击下钻           │
├─────────────────┼────────────────────────────┤
│  ③ 柱状图        │  ④ 散点图                    │
│  各海域月均OHC   │  SST距平 × 风应力             │
│  海域对比        │  海气耦合相关性               │
└─────────────────┴────────────────────────────┘
```

视觉编码依据Mackinlay有效性原则：SST距平（最重要定量维度）映射到地图**位置**与**发散色标亮度**；海域类别映射到**颜色色调**；OHC大小映射到柱状图**长度**；海气耦合关系通过散点图**位置**（双定量轴）呈现。

## 五、交互联动设计
